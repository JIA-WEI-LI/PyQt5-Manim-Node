# Manim Community 節點編輯器
![PyPI - Python Version](https://img.shields.io/pypi/pyversions/Django)

_期待可以做出一個國中生也能輕鬆遊玩並做出屬於自己的數學動畫製作軟體_
## 資料夾說明
* app - 主資料夾
* config - 基礎設定資料夾
  * icon.py - 圖標路徑
  * palette.py - 顏色路徑
* Node - 節點編輯器資料夾
  * nodeEditor_Window.py - 節點編輯器視窗
* resources - 材質包
  * icons - 圖標資料夾
  * qss - QSS 樣式資料夾
